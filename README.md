# rbac-misconfiguration-security-lab
## Objective
Demonstrate how improper role assignments create privilege escalation risks.

## Lab Activities
• Created administrative user without password
• Added guest user to administrator group
• Ran vulnerability scan to detect security issues

## Security Issues Identified
- Privilege escalation
- Weak authentication policy
- Improper RBAC configuration

## Lessons Learned
Role based access control must follow the principle of least privilege.
